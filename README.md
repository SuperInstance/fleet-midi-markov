# 🕸️ fleet-midi-markov

> *Statistical MIDI generation without neural networks*

Builds probability transition matrices from any MIDI training file and generates infinite stylized sequences. The walking bass test from 8 training notes produces: [60,62,64,62,64,65,64,65,64,65,64,62] — pure probability, zero GPUs needed.

---

## 📦 Installation

```bash
# npm
npm install @superinstance/midi-markov

# Docker
docker pull ghcr.io/superinstance/fleet-midi-markov:latest

# Clone
git clone https://github.com/SuperInstance/fleet-midi-markov.git
```
